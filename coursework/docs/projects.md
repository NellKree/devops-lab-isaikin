    ### **Проект 1: Мобильное приложение для составления инвестиционного портфеля**

    **Направление:** Финансовые технологии (FinTech)

    **Описание:**  
    Разработка мобильного приложения, предназначенного для помощи пользователям в формировании и управлении инвестиционным портфелем. Приложение предоставляет инструменты для анализа финансовых инструментов, распределения активов и оценки рисков.  

    Основная цель проекта — упростить процесс принятия инвестиционных решений для начинающих и частных инвесторов. Пользователь может сформировать портфель из различных активов (акции, ETF и другие финансовые инструменты), оценить потенциальную доходность и уровень риска, а также отслеживать структуру портфеля в динамике.

    В рамках проекта были проработаны основные сценарии пользовательского взаимодействия, логика формирования портфеля и методы визуализации инвестиционной информации.

    **Ключевые возможности:**
    - формирование персонального инвестиционного портфеля  
    - анализ распределения активов  
    - базовая оценка риска и потенциальной доходности  
    - визуализация структуры портфеля  
    - удобный интерфейс для отслеживания активов  

    **Технологии:**  
    Мобильная разработка, основы финансовой аналитики, UX/UI дизайн

    ### **Проект 2: PIWO (Perm Is Walking On) — мобильное приложение с маршрутами по Перми**

    **Направление:** Туристические цифровые сервисы / Городские цифровые продукты

    **Описание:**  
    Разработка концепции и прототипа мобильного приложения **PIWO (Perm Is Walking On)**, которое помогает жителям и туристам исследовать город Пермь с помощью интерактивных маршрутов.

    Приложение предлагает пользователю тематические прогулочные маршруты, включающие культурные, исторические и развлекательные точки города. Каждый маршрут представляет собой готовый сценарий прогулки с описанием мест, рекомендациями и дополнительной информацией.

    Основная идея проекта — сделать изучение города более интерактивным и доступным, объединив элементы туризма, развлечений и цифровых сервисов.

    **Ключевые возможности:**
    - готовые туристические и развлекательные маршруты по городу  
    - информация о достопримечательностях и интересных местах  
    - рекомендации по посещению кафе, музеев и общественных пространств  
    - интерактивная карта маршрута  
    - удобная навигация по городу  

    **Технологии:**  
    Мобильная разработка, картографические сервисы, UX/UI дизайн, продуктовый подход

    ### **Проект 3: ML-система обнаружения бот-активности в социальных сетях**

    **Направление:** Data Science / Machine Learning / Анализ социальных сетей

    **Описание:**  
    Разработка системы на основе методов машинного обучения для выявления и анализа бот-активности в социальных сетях.  

    Проект направлен на автоматическое обнаружение подозрительных аккаунтов, которые могут использоваться для накрутки активности, распространения спама или манипуляции общественным мнением.

    Система анализирует различные параметры поведения пользователей: частоту публикаций, характер взаимодействия с контентом, структуру подписок и другие поведенческие признаки. На основе этих данных строится модель классификации, которая определяет вероятность того, что аккаунт является ботом.

    В рамках проекта рассматривались данные из различных платформ, включая:
    - YouTube  
    - Telegram  
    - Twitch  
    - Instagram  

    **Ключевые возможности:**
    - анализ поведения аккаунтов  
    - выявление подозрительных паттернов активности  
    - классификация аккаунтов (бот / реальный пользователь)  
    - масштабируемая архитектура для анализа больших объёмов данных  

    **Технологии:**  
    Python, библиотеки машинного обучения, анализ данных, методы классификации, работа с API социальных сетей
